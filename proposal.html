Final Project Proposal: Self Balancing Robot!

Concept/Motivation

I remember watching a video about this robot dog made by Boston Dynamics designed to patrol construction sites using its cameras and sensors to collect data safely. A cool feature of this robot is that when it’s pushed, it’s able to rebalance itself and continue walking. I wanted to try making a low level version of this by creating a two-wheeled self balancing robot.

Bill of Materials

  • 2 Wheels
  • 2 Geared DC Motors
  • 1 Arduino Controller Board
  • Wires
  • Duct Tape
  • Screws
  • 1 Accelerometer/Gyroscope Module (MPU-6050)
  • 1 Motor Driver Shield
  • 1 Battery (7.4 V lithium)
  • Material for Casing (Plastic, Cardboard, etc.)
  • 1 Switch

Timeline/Contingency Plans

  • Create schematic and gather materials
  • Write the code for the robot
  • Assemble the robot and upload the code
  • This will be done in small steps like testing the motors, testing the gyroscope/accelerometer, creating the casing for the robot (3D modeling maybe), etc.
  • I’ll also be sure to take different safety precautions since we’re working in higher voltages
  • Debug and rebuild until it works
  • If something goes wrong with any materials (short circuit, malfunction, etc.), I will be sure to have some extras on me
